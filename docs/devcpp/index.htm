What is Dev-C++?

"Bloodshed Dev-C++ is a full-featured Integrated Development Environment (IDE) for the C/C++ programming language. It uses Mingw port of GCC (GNU Compiler Collection) as it's compiler." (quote from bloodshed.net)
So, if you have no money to buy Microsoft Visual C++, but you want to compile SPTK-Software under Windows you have found the right website ;-)
